# Depth map

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python depth_map.py --input 'path/to/input/image' --output 'path/to/output/image'
```

Example

```bash
python depth_map.py --input demo/landscape.png --output demo/landscape_depth.png
```
